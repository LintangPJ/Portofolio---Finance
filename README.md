# 💰 Finance Data Analysis Project

## 📌 Project Overview
This project analyzes financial data to predict future trends and customer behavior. The goal is to help businesses make data-driven decisions using **Linear Regression** and **Clustering** techniques.

## 🛠️ Tools & Technologies Used
* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **Scikit-Learn** (Linear Regression, K-Means Clustering)
* **Jupyter Notebook**

## 📊 Key Analysis & Insights
1.  **Sales Prediction:** Built a Linear Regression model to forecast next month's sales based on marketing spend.
2.  **Customer Segmentation:** Used Clustering to group customers into 'High Value', 'Standard', and 'Low Value' based on their transaction history.
3.  **Profit Analysis:** Identified which product categories yield the highest profit margin.
